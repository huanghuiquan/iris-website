今天我完成了个人网站的搭建，使用了以下技术栈：

## 技术选择

### Next.js
选择 Next.js 作为主要框架的原因：
- App Router 提供直观的路由管理
- 出色的性能和 SEO 支持
- 与 Vercel 部署无缝集成

### Tailwind CSS
- 快速原型开发
- 响应式设计支持
- 良好的生态系统

### MDX
- 在 Markdown 中使用 React 组件
- 灵活的博客内容管理
- 支持代码高亮和公式

## 项目结构

```
iris-website/
├── app/              # 页面路由
├── components/       # 可复用组件
├── lib/             # 工具函数
├── content/         # 博客内容
└── public/          # 静态资源
```

## 核心功能

1. **个人介绍页面** - 展示个人信息
2. **博客列表** - 展示所有文章
3. **博客详情** - MDX 渲染文章内容
